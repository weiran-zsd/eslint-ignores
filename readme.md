# eslint-ignores

## Usage

```sh
$ npm install @weiran.zsd/eslint-ignores -D
```

```js
// eslint.config.js
import js from "@eslint/js";
import { ignores } from "@weiran.zsd/eslint-ignores";

export default [js.configs.recommended, ignores()];
```
